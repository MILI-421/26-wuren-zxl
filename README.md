ros学习笔记（图片版）

作业完成思路
```
1.搞清楚作业想要帮助我们熟悉什么知识点和操作——控制乌龟运动，练熟ROS节点通信、自定义消息、参数管理和launch文件
2.观察作业配套文件fsd_common_msgs,思考它的作用，是一个功能包。熟悉标准ROS工作空间结构，知道配套功能包的定位
3.学习通信的流程和所需的文件，要构建一套可闭环运行的乌龟控制ROS系统，需包含python消息发布节点、参数配置文件、C++控制节点、启动脚本
4.上网搜索模板，学习参数的含义，设定参数
5.了解代码启动命令流程，运行调试
```
完成作业遇到的困难
```
1.ros通信模板的语法太多太复杂了，没办法学完自己单独写—
解决办法：先搜索模板，边抄边熟悉。先了解整体的框架逻辑，后面找时间补课
2.写好后发现界面出现了但是乌龟不会动
解决办法：继续研究整体逻辑，发现xxx.py会用.msg发速度命令给节点，且跟着教程在多次检验后发现问题出在python消息发布节点，改进.py文件，后续成功运行
3.虽然完成了作业，但是很依靠vscode的fitten code插件；b站的教学视频还没有来得及看
```
